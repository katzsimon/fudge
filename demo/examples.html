Fudge Options Examples

Showcasing the different possible options.
They are being set via data attributes for easy demonstration, but can be set with JavaScript as well (See README).

Select Search from Select Element

    <select class="form-control fudge">
        <option value="">Please Select</option>
        <option value="Wishbone">Wishbone</option>
        <option value="Ellie">Ellie</option>
        <option value="Chico">Chico</option>
        <option value="Meisie">Meisie</option>
        <option value="Black Meisie">Black Meisie</option>
    </select>

Select Search from Input Element with JSON options

    <input  type="text" class="form-control fudge" value="Fudge"
            data-options='[
                {"value":"","text":"Please Select"},
                {"value":"Deaunan","text":"Deaunan"},
                {"value":"Fudge","text":"Fudge"},
                {"value":"Meisie","text":"Meisie"},
                {"value":"Black Meisie","text":"Black Meisie"},
                {"text":"Smokey"}]'>

Select Add Option from Input Element

    <input type="text" class="form-control fudge" data-add="true"
           data-options="::Please Select,Deunan::Deunan,Fudge,Meisie,Black Meisie,Smokey">

Select Add Option from Select Element

with Showing Target/Original Select
    <select class="form-control fudge" data-fudge-add="true" data-fudge-show_target="true">
        <option value="">Please Select</option>
        <option value="Wishbone">Wishbone</option>
        <option value="Ellie">Ellie</option>
        <option value="Chico">Chico</option>
        <option value="Meisie">Meisie</option>
        <option value="Black Meisie">Black Meisie</option>
    </select>

Tags from Input Element

    <input type="text" class="form-control fudge" data-add="true" data-mode="tags"
           data-options="Deunan::Deunan,Fudge,Meisie,Black Meisie,Smokey">

Tags from Input Element with Tag Limit

    <input type="text" class="form-control fudge" data-add="true" data-mode="tags" data-taglimit="2"
           data-options="Deunan::Deunan,Fudge,Meisie,Black Meisie,Smokey">
